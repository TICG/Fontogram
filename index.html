The above logo is based on two symbols: A and 🎞 (of particular Windows fonts). Fontogram is a method and library for generating a large variety of such icons for Microsoft .NET WPF applications, using a simple scheme for combining WPF Border shapes with symbols of any installed fonts.

The screenshot below shows more Fontogram examples.

One creates an icon by creating an instance of the WPF control class 'Fontogram'. The provided sample code shows how to create custom Fontograms in XAML and how to create reusable Fontogram packs in C# code. Info about Fontogram attributes can be found in the comments for some of the built-in Fontograms: Fontogram.cs.

The library and sample code is completed by the Fontogrammer tool, which makes it easy to design custom Fontograms with their specific creation code.
